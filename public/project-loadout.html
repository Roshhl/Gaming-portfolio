DERMACAST

Build

HTML Tailwind CSS JavaScript OpenWeather API

Difficulty Level

People with different skin types, such as eczema, often do not know how the weather affects their skin. As someone who also suffers from seasonal dry skin in the winter, I thought it would be beneficial to create an app that provides advice tailored to individual skin types. And that is exactly what I did. I built an app using the OpenWeather API and JavaScript that allows users to select their skin type and receive weather based skincare suggestions.

Built using: OpenWeather API, HTML, Tailwind CSS and JavaScript This project helped improve my skills in working with APIs and solidified my understanding of conditional logic in JavaScript, as well as mobile responsive design. It also deepened my understanding of responsive layouts and real world use cases.

PORTFOLIO

Build

HTML Tailwind CSS JavaScript Three JS Figma Blender

Difficulty Level

When it came to deciding and drafting my portfolio, I always knew I did not want to go with the standard design style. I wanted to make my portfolio stand out. I wanted to create a unique experience that was a true reflection of my creative talents and personality. I decided to base my portfolio on an immersive gaming experience. As someone who grew up playing Nintendo Wii games like Mario Kart, I wanted my portfolio to mimic the experience of a gaming console, allowing me to tap into my childhood nostalgia. That is why I included features such as game tiles for each project, a home menu, and a combination of 2D and 3D elements to simulate launching a game when viewing my work.

Built using: Figma, Blender, Tailwind CSS, HTML and Three JS This project introduced me to the integration of 3D design into web interfaces, strengthening my skills in user interface design, animation planning, and layout structure. I also learned the art of storytelling through user experience.
